Installation on your server via FTP

How To Install DOZ Via FTP :
Step 1 - Log into your hosting space via an FTP software (Filezilla for example).
Step 2 - Unzip your Download Pack file and use the extracted DOZ theme folder.
Step 3 - Upload the extracted DOZ theme folder into one folder or at the root of your server.

VEGAS IMAGE OR SLIDESHOW - SET UP

VEGAS is a jQuery plugin which adds beautiful animated background slideshows to your page body or any of its elements. It works on IE9+, Safari, Firefox and Chrome.
